1. DESCRIPTION
-------------------------------------------------------------------------------
The MultiReplace plugin list files to find and replace using Notepad++:

* Reformat cpp files.

* Find and replace in the [Total Commander](https://www.ghisler.com/) button
  bar files to renumber the records.

* Decorate and strip changelog files from
  [Changelog files organization](https://gist.github.com/andry81/d278e6d129ca1af326eafb67470a2ae3)
  in [WinMerge](https://winmerge.org/).

* Support (tacklebar)[https://github.com/andry81/tacklebar] file path lists
  edit.

* Find and replace in batch files from `contools` project.
